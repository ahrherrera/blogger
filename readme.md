# How To Install

To set up the project you need to:
-  Rename `.env.example` and configure each environment variable
-  `composer install`
-  `php artisan key:generate`
-  `php artisan migrate`
-  `php artisan db:seed`
-  `php artisan serve` (**Skip this step if you're using Laragon**)


- Use PHP 7.4
- Laravel 5.7.*
- MySQL Server
